# UI Sound Effects

This folder contains sound effects for UI interactions.

## Structure

```
sounds/
├── ui/
│   ├── click.mp3        # Button clicks
│   ├── hover.mp3        # Hover effects
│   ├── toggle-on.mp3    # Switch on
│   ├── toggle-off.mp3   # Switch off
│   ├── checkbox.mp3     # Checkbox toggle
│   ├── modal-open.mp3   # Modal opening
│   ├── modal-close.mp3  # Modal closing
│   ├── notification.mp3 # Notifications/alerts
│   ├── success.mp3      # Success feedback
│   ├── error.mp3        # Error feedback
│   └── scroll.mp3       # Scroll interactions
└── README.md
```

## Recommended Formats

- **MP3**: Best compatibility
- **WAV**: Higher quality, larger files
- **OGG**: Good compression, not supported in Safari

## Recommended Sources

- [Freesound.org](https://freesound.org) - Free CC sounds
- [Pixabay](https://pixabay.com/sound-effects) - Free sounds
- [Soundsnap](https://www.soundsnap.com) - Premium sounds
- [UI8](https://ui8.net) - Premium UI sounds

## Tips

- Keep sounds short (< 500ms for clicks/hovers)
- Use subtle sounds that don't annoy users
- Normalize volume levels across all sounds
- Consider offering a mute option in settings
